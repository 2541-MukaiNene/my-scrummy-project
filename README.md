## ターゲット（誰のため？）
勉強や作業で疲れていたり、夜ひとりで気分が落ち込みやすい人
## 解決したい課題
夜に部屋が暗いと気分まで沈みやすかったり、課題や勉強で疲れてテンションが下がってしまうことがある。
そんなときに、自動で空間を明るくしながら、気分までちょっと上げてくれるデバイスがあればいいと思った。
## 使用する3つのセンサー・部品
① 光センサー（部屋の明るさを測る）
② 音センサー（声の大きさや元気度を測る）
③ ボタン（「励まして！」ボタン）
④ RGB LED（虹色ライト）
⑤ ブザー（明るい効果音）
## デバイスの動作論理（ロジック）
「もし、部屋が暗くて、さらに 声が小さい（元気がなさそう）、または 励ましてボタンが押されたら、虹色LEDを点灯させて、明るい音を鳴らす。」
